更新履歴

はじめに

新機能

バージョン履歴とリリースノート。

v1.9.0

審査中

  • インライン数式トグル $...$ のレンダリングを制御する新設定（デフォルト: OFF）。LaTeX標準記法 \(...\) と \[...\] は常に動作
  • 数式拡大ボタン 数式ブロックの🔍をクリックで拡大表示。Mermaid図と同様の操作感
  • クロスプラットフォームショートカット プラットフォーム固有のキー記号：Mac（⌘⇧P、⌘⌥F）vs Windows/Linux（Ctrl+Shift+P、Ctrl+H）
  • 数式コードブロック ```math、```latex、```mathematica 対応。言語セレクター、Edit/Hide切替、拡大ボタン搭載
  • コピーボタン修正 誤ったコードブロックをコピーする問題を修正。インデックスベースからコンテンツベースのマッチングに変更

v1.8.0

リリース済

  • エディタ強化 行移動（Alt+↑/↓）、日付挿入（Ctrl+;）、検索・置換（Ctrl+H、正規表現）、コマンドパレットでテキストケース変更
  • 自動リンク化 プレーンテキスト内のURL/メールアドレスが自動的にリンクに。コードブロックと既存Markdownリンクは除外
  • 数式ブロックサポート GitHub/GitLab互換の ```math コードブロックが数式としてレンダリング。$$...$$ 記法と併用
  • 大きいファイルの可視化 ファイルサイズで色分け：黄色（>100KB）、オレンジ（>200KB）、赤（>300KB）
  • TOCスクロール改善 2段階スクロール：長距離（>1000px）は瞬時にジャンプ、残り500pxをスムーズに。スクロール後の位置検証で精度向上
  • ファイル名変換ロジック統一 Git pushとExportで統一。Windows予約名を自動処理、変換時はトースト通知、100文字で切り詰め
  • クリーンMarkdownコピー コピー時にHTMLタグなしのクリーンテキストを出力。メモ帳、Sublime Text、ChatGPT、Grokと互換
  • LaTeX自動変換 AIツールからの \(...\) → $...$、\[...\] → $$...$$ を自動変換（入力・ペースト両方）

v1.7.0

リリース済

  • キーボードファースト設計 Ctrl+Shift+1/2でNotes/TOCパネル切替、矢印キーでリスト操作、Enter/Space/Escapeでモーダル操作
  • Zen Mode廃止 F11（ブラウザ全画面）+ Alt+F（Focus Mode）で代替 — 2.76KBバンドル削減
  • Git同期状態の可視化 起動時自動Fetch、「N件のリモート更新」バナー、Trash同期バッジ（✓同期済/⚠要同期/ローカル）
  • アニメーション最適化 50-200ms範囲に最適化 — カーソルフラッシュ、TOCハイライト、見出しハイライトが瞬時に反応
  • E2Eテスト Playwrightで19件のテスト追加（コア機能、エッジケース、キーボードショートカット）

v1.6.0

リリース済

  • Command Palette VS Code風パレット、50+コマンド — Ctrl+Shift+P
  • Focus Mode タイプライタースクロールと3段階dimmingで集中執筆
  • 起動時間95%改善 Core Web Vitals合格 — 初期バンドル: 698KB → 84KB
  • Archive表示70%高速化 軽量レンダラー採用 — 100KBドキュメントが1秒以内にロード
  • TOC拡張機能 階層折りたたみ（20見出し以上）、検索バー（15見出し以上）、キーボードナビ

v1.5.0

リリース済

  • 大規模ドキュメント対応 100KB超ファイルの読み込みが57%高速化
  • Archiveピン留めエクスポート Archiveフォルダのピン留めノートのみを.zipで出力
  • Side Panel Sticky Toolbar Side Panelでスクロールしてもツールバーが常に表示
  • エディタ機能強化 コールアウト、ページ内リンク、LaTeX数式、Mermaid拡大表示

v1.4.0

リリース済

  • 目次（TOC）の強化 スクロール中に現在セクションをハイライト、ナビゲーション時の視覚的フィードバック
  • データ安全性の向上 同時編集による消失を防ぐ3層防御で自動保存の信頼性向上
  • UX改善 スムーズなトランジション、コードブロック高さ制限、スター/ピンボタン統合

v1.3.0

リリース済

  • Git連携の強化 Fetch機能追加、Pull/Sync操作の競合検出を改善
  • 重複ノート管理 同名ノートをside-by-side比較、保持するノートを選択
  • スマート検索の強化 複数キーワードAND検索、ハイライト、フォルダバッジ表示

v1.2.0

リリース済

  • ツールバー刷新 ホバーで開くメニュー、ワンクリック減でアクセス

  • Overview Ruler VSCode風の差分マップ、クリックでジャンプ
  • Auto Pair 括弧・引用符の自動補完
  • ノート検索 サイドバーで入力ごとにフィルタリング
  • ワイドスクリーン対応 1600px以上でサイドバー・TOCがコンテンツ外に
