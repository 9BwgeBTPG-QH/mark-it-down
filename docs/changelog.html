What's New

Version history and release notes.

v1.8.0

Under Review

  • Editor Enhancements Line move (Alt+↑/↓), date insertion (Ctrl+;), Find & Replace (Ctrl+H with regex), text case change via Command Palette
  • Auto-Linking URLs and emails in plain text become clickable links automatically. Excludes code blocks and existing Markdown links
  • Math Block Support GitHub/GitLab-compatible ```math code blocks render as equations. Works alongside $$...$$ syntax
  • Large File Visualization Files color-coded by size: Yellow (>100KB), Orange (>200KB), Red (>300KB)
  • Improved TOC Scrolling Two-phase scroll for distant headings: instant jump (>1000px), smooth finish (last 500px). Post-scroll verification ensures accurate positioning
  • File Name Sanitization Unified conversion logic for Git push and Export. Handles Windows reserved names, shows toast notifications, truncates to 100 characters
  • Copy as Clean Markdown Copy outputs clean text (no HTML tags), compatible with Notepad, Sublime Text, ChatGPT, Grok
  • LaTeX Auto-Conversion Converts \(...\) → $...$ and \[...\] → $$...$$ from AI tools (input and paste)

v1.7.0

Released

  • Keyboard-First Design Ctrl+Shift+1/2 for Notes/TOC panels, arrow keys for list navigation, Enter/Space/Escape for modals
  • Zen Mode Removed Use F11 (browser fullscreen) + Alt+F (Focus Mode) instead — 2.76KB bundle reduction
  • Git Sync Status Visibility Auto-fetch on startup, "N remote updates" banner, Trash sync badges (✓Synced/⚠Sync/Local)
  • Snappier Animations Optimized to 50-200ms range — cursor flash, TOC highlight, heading highlight all feel instant
  • E2E Testing 19 Playwright tests covering core functionality, edge cases, and keyboard shortcuts

v1.6.0

Released

  • Command Palette VS Code-style palette with 50+ commands — Ctrl+Shift+P
  • Focus Mode Distraction-free writing with typewriter scroll and 3-level dimming
  • 95% Faster Startup Core Web Vitals PASS — Initial bundle: 698KB → 84KB
  • 70% Faster Archive Lightweight renderer for Archive notes — 100KB docs load in under 1s
  • Enhanced TOC Collapsible headings (20+), search bar (15+), keyboard navigation

v1.5.0

Released

  • Large Document Support 57% faster loading for 100KB+ files with smart optimizations
  • Archive Pinned Export Export only pinned notes in Archive folder as .zip
  • Side Panel Sticky Toolbar Toolbar stays visible while scrolling in Side Panel
  • Enhanced Editor Callouts, page anchors, LaTeX math, Mermaid zoom

v1.4.0

Released

  • Smarter Table of Contents Highlights current section while scrolling, visual feedback on navigation
  • Enhanced Data Safety 3-layer protection against concurrent edits for reliable auto-save
  • UX Polish Smooth transitions, code block height limits, unified star/pin button

v1.3.0

Released

  • Enhanced Git Integration Fetch support, improved conflict detection for Pull/Sync
  • Duplicate Note Management Side-by-side comparison to choose which note to keep
  • Smarter Search Multi-keyword AND search, highlights, folder badges

v1.2.0

Released

  • Refreshed Toolbar Hover-activated menus for faster access
  • Smarter Git Sync Better conflict detection with 4-level risk indicators
  • Overview Ruler VSCode-style diff map with click-to-jump
