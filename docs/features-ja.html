Mark It Down

機能

はじめに

Markdownで書く。

AI非対応。プラグインなし。
1ペイン。書いて、すぐ見る。

2つのモード

New Tab — 全画面で

専用ワークスペース。Zenモード利用可能（F11）。

Side Panel — Webの横で

AIの横で書く。ブラウジングしながら書く。

両方開ける。同期しない。

更新前を見ながら、書き換える。

Markdown

GitHub Flavored Markdown

テーブル、タスクリスト、シンタックスハイライト。

Mermaid図

フローチャート、シーケンス図、円グラフ。クリックで拡大。

数式（LaTeX）

インライン $E=mc^2$ とブロック $$...$$ 形式。

コールアウト

GitHub Alerts: NOTE、TIP、IMPORTANT、WARNING、CAUTION。

表示

目次（TOC）

長文ドキュメント用の自動生成目次。

ダーク/ライトモード

環境に合わせてテーマ切替。

Auto Pair

括弧・引用符・バッククォートを自動補完。

ノート管理

5つのフォルダ。固定。

Inbox / Template / Archive / Trash / System。

テンプレート & ピン

テンプレートを保存。重要なノートを上部に固定。

ドラッグ&ドロップ

フォルダ間でノートを移動。

ソート&検索

名前順または日付順。インクリメンタル検索。

保存

自動保存

保存ボタンなし。ただ書くだけ。

Chrome拡張機能ストレージ

ブラウザキャッシュクリアでも残る。

バックアップ

すべてを.zipでエクスポート。またはGit同期。

フェイルセーフ削除

同期済みノートは3段階で削除。

Git同期

GitHub / GitLab

Personal Access Tokenで接続。

手動コミット

同期タイミングはあなたが決める。

Pull & マージ

競合を並べて解決。

差分 & 履歴

バージョン比較。過去20件のコミットを表示。

デュアルモード

New TabとSide Panelを同時に開く。

独立したウィンドウ

片方で編集。もう片方で読む。

競合解決

Merge Changesボタンが点滅。選択して保持。

比較ツール

スクロール同期、差分ハイライト、TOCナビゲーション。
