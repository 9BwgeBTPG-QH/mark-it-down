Mark It Down

機能

はじめに

Markdownで書く。

AI非対応。プラグインなし。
1ペイン。書いて、すぐ見る。

2つのモード

New Tab — 全画面で

専用ワークスペース。Focus Mode利用可能（Alt+F）。

Side Panel — Webの横で

AIの横で書く。ブラウジングしながら書く。

固定ツールバー

Side Panelでスクロールしてもツールバーが常に表示。素早い操作が可能。

ブラウザ全画面（F11）

F11でブラウザ標準の全画面表示。全ブラウザで動作。

Focus Mode

Alt+Fで切替。タイプライタースクロールでカーソルを中央維持。テキストdimmingで集中ゾーンを強調。

両方開ける。同期しない。

更新前を見ながら、書き換える。独立したウィンドウで同じデータを共有。

Markdown

GitHub Flavored Markdown

テーブル、タスクリスト、シンタックスハイライト。

Mermaid図

フローチャート、シーケンス図、円グラフ。クリックで拡大。

数式（LaTeX）

インライン $E=mc^2$ とブロック $$...$$ 形式。

コールアウト

GitHub Alerts: NOTE、TIP、IMPORTANT、WARNING、CAUTION。ライト/ダークモードで色分け表示。

ページ内リンク

内部リンク: [テキスト](#見出し) でページ内ジャンプ。GitHub互換の見出しID生成。

画像

外部URLのみ（![alt](https://...)）。ストレージ制限のため直接埋め込み非対応。

表示

目次（TOC）

自動生成目次。スクロール中に現在セクションをハイライト。クリックでジャンプ。競合解決でもTOCナビゲーション。階層折りたたみ（20見出し以上）、検索バー（15見出し以上）、キーボードナビゲーション。

ダーク/ライト/システム連動テーマ

設定でテーマ切替。システム連動でOS設定に従う。

スムーズなトランジション

ノート切替時に200msのフェード効果。コードブロックは20行超でスクロール可能（max-height: 400px）。

Auto Pair

括弧・引用符・バッククォートを自動補完。設定でON/OFF可能。

コードブロック高さ制限

長いコードブロック（20行超）はスクロール可能。CopyボタンはMarkdownソースから全コンテンツをコピー。

ノート管理

5つのフォルダ。固定。

Inbox（作業中）/ Template（再利用用）/ Archive（ロック）/ Trash（削除待ち）/ System（読み取り専用ガイド）。カスタムフォルダは設計上非対応。

テンプレート & ピン

Templateフォルダにテンプレートを保存。重要なノートを任意のフォルダの上部に固定（★ボタン）。Alt+Pショートカット。

ドラッグ&ドロップ

フォルダ間でノートを移動。フォルダエリア全体（フォルダ名 + ノートリスト）にドロップ可能。

ソート&検索

名前順または日付順（時計アイコンで切替）。サイドバーでインクリメンタル検索（1文字入力 = 即時フィルタ）。マッチするフォルダは自動展開。

重複ノート検出

同じタイトルのノート検出時に警告バナー表示。side-by-side比較で保持選択。

サイドバーリサイズ

ドラッグで幅調整。ダブルクリックでリセット。Slack風の操作感。

保存

自動保存

保存ボタンなし。入力後0.5秒で自動保存。レースコンディション対策でデバウンス処理。

大規模ドキュメント

100KB超のMarkdownファイルに対応。CodeMirror自動無効化。ローディングスケルトンでUX向上。CSS最適化で快適に編集。Archive軽量レンダラー（70%高速化）。

Archiveフォルダで軽くなる理由

Archiveフォルダのノートは、通常のProseMirrorエディタではなく軽量な静的HTMLレンダラーを使用します。Archiveノートは読み取り専用のため、編集機能（Undo/Redo、入力監視など）をスキップでき、70%高速化（100KB超ドキュメントで3秒以上→1秒以下）を実現しています。

Chrome拡張機能ストレージ

ブラウザキャッシュクリアでも残る。拡張機能アンインストールまでデータ保持。unlimitedStorage権限。

インポート/エクスポート

ドラッグ&ドロップで.md/.zipファイルをインポート。エクスポート: 単一.md、全ノート.zip、Archiveピン留め.zip。

バックアップ

すべてを.zipでエクスポート。またはGit同期でバージョン履歴管理。

Archiveピン留めエクスポート

Archiveフォルダでピン留めしたノートのみを.zipで一括エクスポート。完成した重要なノートにピン留めを付けて、「Export → .zip (Archiveピン留め)」でまとめてバックアップできます。重要な完成品の定期バックアップに便利です。

フェイルセーフ削除

同期済みノート: Trash → Sync → Delete（3段階）。未同期ノート: Trash → Delete（2段階）。Trash内に同期ステータスバッジ表示。

ストレージ修復&分析

設定 → Troubleshooting: Repair Storage（重複検出）、Storage Analysis（使用状況、一括アーカイブ）。

Git同期

GitHub / GitLab

Personal Access Tokenで接続（Fine-grained PAT推奨）。AES-GCM暗号化で安全に保存。

意図的なバージョン管理

OneDriveのような自動同期ではない。手動コミット = 意味のあるスナップショット。ローカル自動保存（履歴なし）vs リモートコミット（履歴あり）。

Commit & Sync

真の同期: Commit → Pull → 競合解決 → Push。カスタムコミットメッセージ。バッチコミット（全変更を1コミットに統合）。

Pull & マージ

競合を並べて解決。リスクレベルインジケーター（Low/Medium/High/Critical）。Myers差分アルゴリズム。TOCナビゲーション。スクロール同期。

Git Fetch

ローカルを変更せずにリモート状態を確認。起動時自動Fetch。リモート更新時にバナー表示。

差分 & 履歴

バージョンを並べて比較。過去20件のコミットを表示。Overview Rulerで差分ナビゲーション。

ファイル追跡

フォルダ移動・タイトル変更時に古いファイルを自動削除。リモートの重複ファイルを防止。

Discard Changes

ローカル変更を最後のPush時点に戻す。Git Settings → Danger Zone。

システムノート

Git同期から除外（ローカルのみ）。Trash + Syncでリモートから削除。アップグレード時に自動クリーンアップ。

デュアルモード

New TabとSide Panelを同時に開く。

独立したウィンドウ

片方で編集。もう片方で読む。

競合解決

両方のウィンドウで同じノートを編集するとMerge Changesボタンが点滅。選択して保持。Git Pull時の競合も同じUIで操作。

比較ツール

スクロール同期、差分ハイライト（+緑/−赤）、Overview Ruler（クリックでジャンプ）、TOCナビゲーション、クイックスクロールボタン。

レスポンシブ競合UI

Side Panel/狭い画面: 単一ペインビュー、大きなボタン。キーボードナビゲーション（矢印キー、Escape）。
