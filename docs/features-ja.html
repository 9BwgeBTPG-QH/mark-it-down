Mark It Down

機能

はじめに

Markdownで書く。

AI非対応。プラグインもなし。1ペイン。
書いた瞬間に見える。それだけ。

2つのモード

New Tab — 全画面で

専用ワークスペース。Zenモード利用可能（F11）。

Side Panel — Webの横で

AIの横で書く。ブラウジングしながら書く。

両方開ける。同期しない。

更新前を見ながら、書き換える。

エディタ

GitHub Flavored Markdown

テーブル、タスクリスト、シンタックスハイライト付きコードブロック。

Mermaid図

フローチャート、シーケンス図、円グラフ。🔍ボタンで拡大表示。

数式（LaTeX）

インライン $E=mc^2$ とブロック $$...$$ 形式。

コールアウト

GitHub Alerts: NOTE、TIP、IMPORTANT、WARNING、CAUTION。

目次（TOC）

長文ドキュメント用の自動生成目次。サイドパネルまたはドロップダウン。

ダーク/ライトモード

環境に合わせてテーマ切替。

Auto Pair

括弧・引用符・バッククォートを自動補完。設定でON/OFF可能。

ノート管理

5つのフォルダ。固定。

Inbox / Template / Archive / Trash / System。すべてに居場所がある。

テンプレート

ノートテンプレートを保存。複製して新規作成。

ピン留め

重要なノートを各フォルダの最上部に固定。

ドラッグ&ドロップ

フォルダ間でノートを移動。シンプルで直接的。

ソート&検索

名前順または日付順。インクリメンタル検索で何でも見つかる。

ツールバー

スクロールに追従するスティッキーツールバー。情報フロー: Entry → Work → Move → Exit。

Entry: Import、Paste

.mdや.zipファイルをドラッグ。またはクリップボードから貼り付け。

Work: Undo、Redo

Ctrl+Z / Ctrl+Y。安心して編集。

Move: フォルダ間移動

Pin、Template、Archive、Trash、Restore。すべて1つのメニューに。

Exit: Export、Copy

.mdとしてエクスポート。クリップボードにコピー。他のツールへ移行。

保存

自動保存（0.5秒）

保存ボタンなし。ただ書くだけ。

Chrome拡張機能専用ストレージ

ブラウザキャッシュクリアでも残る。起動は一瞬。

ファイルシステムではない

エクスプローラーでは見えない。.mdファイルとして取り出すにはExport。

バックアップ: ExportまたはGit

すべてを.zipでエクスポート。またはGitHub/GitLabに同期。

フェイルセーフ削除

同期済みノートは3段階で削除。Trash → Sync → 完全削除。

Git同期

OneDriveのような自動同期ではない。意図的なバージョン管理。同期タイミングはあなたが決める。完全な履歴。誤った上書きなし。

GitHub / GitLab

Personal Access Tokenで接続。パブリックまたはプライベートリポジトリ。

手動コミット

同期タイミングはあなたが決める。カスタムまたは自動生成メッセージ。

Pull & マージ

リモートの変更を取得。競合を並べて解決。

スマート競合解決

空白差分を自動正規化。リスクレベル表示。Myers差分アルゴリズム。

差分表示

同期前にローカルとリモートを比較。

コミット履歴

過去20件のコミットを表示。何が変わったか確認。

暗号化トークン

AES-GCM暗号化。トークンは平文保存されない。

デュアルモード

New TabとSide Panelを同時に開く。

独立したウィンドウ

片方で編集。もう片方で読む。自動同期なし。

競合解決

Merge Changesボタンが点滅。どちらを保持するか選択。

並べて比較

両方のバージョンを表示。手動で決定。

比較ツール

スクロール同期、差分ハイライト、クイックスクロール、TOCナビゲーション。
