Mark It Down

Features

Introduction

Write in Markdown.

No AI. No plugins. Just one pane.
Write, see instantly. That's it.

Two Modes

New Tab — Full screen

Dedicated workspace. Zen mode available (F11).

Side Panel — Beside the web

Write alongside AI. Write while you browse.

Open both. No sync.

See the old version while editing the new one.

Editor

GitHub Flavored Markdown

Tables, task lists, code blocks with syntax highlighting.

Mermaid diagrams

Flowcharts, sequence diagrams, pie charts. Click to zoom.

Math (LaTeX)

Inline $E=mc^2$ and block $$...$$ equations.

Callouts

GitHub Alerts: NOTE, TIP, IMPORTANT, WARNING, CAUTION.

Table of Contents

Auto-generated TOC for long documents. Side panel or dropdown.

Dark/Light mode

Switch themes to match your environment.

Auto Pair

Brackets, quotes, backticks auto-complete. Toggle in settings.

Note Management

Five folders. Fixed.

Inbox / Template / Archive / Trash / System. Everything has a place.

Templates

Store note templates. Duplicate to start fresh.

Pin notes

Keep important notes at the top of each folder.

Drag & drop

Move notes between folders. Simple and direct.

Sort & search

Name or date order. Incremental search to find anything.

Toolbar

Sticky toolbar follows as you scroll. Information flow: Entry → Work → Move → Exit.

Entry: Import, Paste

Drag .md or .zip files. Or paste from clipboard.

Work: Undo, Redo

Ctrl+Z / Ctrl+Y. Edit with confidence.

Move: Between folders

Pin, Template, Archive, Trash, Restore. All in one menu.

Exit: Export, Copy

Export as .md. Copy to clipboard. Move to other tools.

Storage

Auto-save (0.5 seconds)

No save button. Just type.

Chrome extension storage

Survives browser cache clear. Instant load.

Not filesystem

Not visible in Explorer. Use Export to get .md files.

Backup: Export or Git

Export all as .zip. Or sync to GitHub/GitLab.

Failsafe deletion

Synced notes need 3 steps to delete. Trash → Sync → Delete permanently.

Git Sync

Not auto-sync like OneDrive. Intentional version control. You control when to sync. Full commit history. No accidental overwrites.

GitHub / GitLab

Connect with Personal Access Token. Public or private repo.

Manual commits

You decide when to sync. Custom or auto-generated messages.

Pull & merge

Fetch remote changes. Resolve conflicts side-by-side.

Smart conflict resolution

Auto-normalize whitespace. Risk level indicator. Myers diff algorithm.

Diff viewer

Compare local vs remote before syncing.

Commit history

View past 20 commits. See what changed.

Encrypted tokens

AES-GCM encryption. Tokens never stored in plain text.

Dual Mode

Open New Tab and Side Panel simultaneously.

Independent windows

Edit in one. Read in the other. No auto-sync.

Conflict resolution

Merge Changes button blinks. Choose which to keep.

Side-by-side comparison

See both versions. Decide manually.

Comparison tools

Sync scroll, diff highlight, quick scroll, TOC navigation.
