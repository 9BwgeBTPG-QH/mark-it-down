Features

Introduction

Write in Markdown.

No AI. No plugins. Just one pane. Write, see instantly.

Two Modes

New Tab — Full screen

Dedicated workspace. Zen mode available (F11).

Side Panel — Beside the web

Write alongside AI. Write while you browse.

Open both. No sync.

See the old version while editing the new one.

Editor

GitHub Flavored Markdown

Tables, task lists, code blocks with syntax highlighting.

Mermaid diagrams

Flowcharts, sequence diagrams, pie charts. Click to zoom.

Math (LaTeX)

Inline $E=mc^2$ and block $$...$$ equations.

Callouts

GitHub Alerts: NOTE, TIP, IMPORTANT, WARNING, CAUTION.

Table of Contents

Auto-generated TOC for long documents.

Dark/Light mode

Switch themes to match your environment.

Auto Pair

Brackets, quotes, backticks auto-complete.

Note Management

Five folders. Fixed.

Inbox / Template / Archive / Trash / System.

Templates & Pin

Store templates. Pin important notes to the top.

Drag & drop

Move notes between folders.

Sort & search

Name or date order. Incremental search.

Storage

Auto-save

No save button. Just type.

Chrome extension storage

Survives browser cache clear.

Backup

Export all as .zip or sync to GitHub/GitLab.

Failsafe deletion

Synced notes need 3 steps to delete.

Git Sync

Intentional version control. You control when to sync.

GitHub / GitLab

Connect with Personal Access Token.

Manual commits

You decide when to sync.

Pull & merge

Resolve conflicts side-by-side.

Diff viewer

Compare local vs remote before syncing.

Commit history

View past 20 commits.

Encrypted tokens

AES-GCM encryption. Never stored in plain text.

Dual Mode

Open New Tab and Side Panel simultaneously.

Independent windows

Conflict resolution

Comparison tools

Sync scroll, diff highlight, TOC navigation.
