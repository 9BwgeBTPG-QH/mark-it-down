Mark It Down

ブラウザが、書斎になる。

Mark It Down — AI 出力や Web 記事を咀嚼するための Chrome 拡張 Markdown エディタ ＆ Web クリッパー

取り込む。
必要な部分を書き直す。
次へ送り出す。

Chrome 拡張の Markdown エディタ ＆ Web クリッパー。AI 出力、Web 記事、RSS、断片メモを扱う作業場です。

すべてのノートをブラウザに置き続ける必要はありません。
Mark It Down は、素材が自分の言葉になるまでの静かな場所です。
できあがったら、Export、Git、長期保存先へ送り出します。

Philosophy

哲学

  • まず取り込む。分類はあとで。

    クリップ、フィード、貼り付けた下書きは Inbox へ。
    分類は、読んでからでも間に合います。

  • 保存じゃなくて、咀嚼。

    コピペでは自分のものにならない。
    書き直すことで自分の言葉になる。

  • ローカルに保存し、意図して送り出す。

    下書きは自動保存で手元に残ります。
    Export と Git 同期は、ノートの準備ができたときに実行します。

Workflow

Entry → Edit → Clear → Exit

情報はブラウザに溜め込むだけでなく、通過させることもできます。

  • Entry

    AI出力を貼り付ける。
    ページをクリップする。RSSを保存する。空白から書く。

  • Edit

    自分の言葉で。
    自分のものにする。

  • Clear

    終わったものはArchiveへ。
    繰り返す型はTemplateへ。不要なものはTrashへ。

  • Exit

    PDF、DOCX、HTML、EPUB、LaTeX、.md、Git。
    できあがったノートを、次の場所へ送ります。

Use Cases

実際の使い方

  • Xを記事でもスレッドでも取り込む X / Twitterの記事でもスレッドでも、右クリックの取り込みで即Markdownに変換して保存。
  • 複数環境の日次ログ ノートのタイトルを日付（例 YYYYMMDD）にして、メモは /now の時刻見出しで追記。別の端末で編集しても、Git同期が競合を検出し「両方を結合」で1つにまとめられる。
  • 横で読みながらメモ、タブで集中 Webを見ながらSide Panelでメモ。閉じてNew Tabで同じノートを開き、じっくり編集。
  • 思いつくまま書いて並べ替え リストを思いつくまま書いて、Alt+↑ / ↓ で行を並べ替えて整える。
What's New

最近の追加機能

v2.2.6 — アドレスバーから直接ノートへ。図を拡大して確かめる。長文を分割して育てる。

  • アドレスバーからノートを開く Chrome のアドレスバーで mid と入力してスペースを押すと、ノートをタイトルで検索できる。Enter で新規タブに開く。マウス不要。
  • Today テンプレート Template フォルダに「Today」というノートを置くと、今日のノートがそのひな型で始まる。1つのノートが毎日のレイアウトになる。
  • Mermaid 図を拡大・ズーム・SVG 保存 Mermaid 図をクリックすると拡大表示され、Ctrl+スクロールで 25〜400% ズームできる。そのまま SVG ファイルとして保存可能。
  • 長いノートをシリーズに分割 見出し単位でノートを分割してシリーズにまとめられる。Alt+PageUp/PageDown で移動、結合、シリーズ全体のエクスポートも可能。
  • Web Clipper：ルビを青空文庫記法に変換 ルビ付きのページ（<ruby> タグ）を取り込むと、漢字《かんじ》 形式に変換される。脚注や表を含む長文の取り込み品質も改善した。
FAQ

よくある質問

無料で使えますか？

はい、完全無料です。広告なし、有料プランなし、アカウント登録も不要です。

オフラインで使えますか？

はい、ほぼすべての機能がオフラインで動作します。ノートの作成、編集、検索はすべてオフラインで可能。Git同期のみインターネット接続が必要です。

ブラウザのキャッシュを消すとノートも消えますか？

いいえ、消えません。ノートは拡張機能専用のストレージに保存されており、ブラウザキャッシュとは別です。データは安全です。
