はじめに

貼り付けて、比較して、書き直す。それが「咀嚼」。
保存する場所じゃない。咀嚼する場所。

哲学

  • 道具を減らして、余白を増やす。

    プラグインもウィジェットもない。
    空白のページとあなたの思考だけ。

  • 保存じゃなくて、咀嚼。

    コピペでは自分のものにならない。
    書き直すことで自分の言葉になる。

  • あなたの言葉で、あなたのリズムで。

    サイドパネルでも新しいタブでも。
    集中したいときはFocus Mode。

Entry → Edit → Clear → Exit

情報は溜めるんじゃない、流す。

  • Entry

    AIから貼り付け。
    インポート。
    新規作成。

  • Edit

    自分の言葉で。
    自分のものにする。

  • Clear

    Archive、Template、Trashへ。

  • Exit

    書いたものは、あなたと一緒に。

スクリーンショット

v2.0.0 の新機能

現在審査中

  • Export Suite — PDF・DOCX・PNG・HTML 書いたMarkdownをブラウザの外へ。PDFにはページ番号と目次、DOCXには画像・Mermaid図・数式をすべて埋め込み。PNGは高解像度キャプチャ。HTMLは3テンプレートでネット不要の1ファイル完結型。
  • ツールバー全面リデザイン L1（基本操作）とL2（···で展開）の2層構造に統一。全アイコンをLucideに統一し、コマンドパレットに22コマンドを追加。
  • Cross-Instance Compare & Edit New TabとSide Panelで同じノートを開いても、片方の編集が消えない。差分を並べて比較し、どちらを採用するか選択できる。
  • テーブルUX全面改善 ツールバーが9→17ボタンに拡張。列ソート、CSV/Markdownコピー、Layout Toggle（自動/全幅/均等）、行/列クロスハイライト。
  • 脚注・Callout強化・使い方ヒント 脚注が復活（クリックでジャンプ）。CalloutタイプをCtrl+Shift+Aで循環切り替え。空のノートにランダムTipsを表示。

よくある質問

無料で使えますか？

はい、完全無料です。広告なし、有料プランなし、アカウント登録も不要です。

オフラインで使えますか？

はい、ほぼすべての機能がオフラインで動作します。ノートの作成、編集、検索はすべてオフラインで可能。Git同期のみインターネット接続が必要です。

ブラウザのキャッシュを消すとノートも消えますか？

いいえ、消えません。ノートは拡張機能専用のストレージに保存されており、ブラウザキャッシュとは別です。データは安全です。
