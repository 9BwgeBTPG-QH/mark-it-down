Storage Repair

If duplicate notes or data inconsistencies occur, you can repair storage from the settings.

How to Repair Storage

  1. Open ‚öô Settings ‚Üí Troubleshooting tab ‚Üí Repair Storage
  2. Click the "Scan" button
  3. If issues are found, select notes to delete and repair

What Gets Detected

Item Description
Duplicate notes Notes with the same title exist in multiple folders

üí° Rarely, ghost notes (notes that exist in storage but don't appear in the UI) may be created if Git sync is interrupted. This feature can detect and delete them.

Storage Analysis

Check storage usage and organize old notes in bulk.

How to Analyze Storage

  1. Open ‚öô Settings ‚Üí Troubleshooting tab ‚Üí Storage Analysis
  2. Click the "Analyze" button

What's Displayed

  • Notes by folder - Number of notes and percentage per folder
  • Top 10 largest notes - With checkboxes for bulk archive
  • Stale notes - Notes not updated for 90+ days

Bulk Archive

Select notes and move them to Archive folder in bulk with pinning. Useful for regular backups and storage cleanup.

üí° System Notes are excluded from analysis (auto-generated guide files).

Duplicate Note Detection

If notes with the same title are detected, a warning banner is displayed with a side-by-side comparison to choose which one to keep.

What Happens

  • A warning banner appears when duplicate notes are detected
  • Click to open a side-by-side comparison modal
  • Choose which note to keep
