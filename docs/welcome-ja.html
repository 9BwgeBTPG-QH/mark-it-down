インストールありがとうございます

新しいタブが Markdown ワークスペースになりました。

Chrome がダイアログを表示したら

「そのままにする」をクリックすると、新しいタブで Mark It Down が起動します。

「このページを変更するつもりでしたか？」→ 「そのままにする」

気が変わったら「元に戻す」をクリック、または拡張機能を削除すればいつでも元の新しいタブに戻ります。

Ctrl + T で新しいタブを開いてみてください。
Web Clipper・Markdown エディタ・サイドパネル ── ブラウザだけで完結するノートアプリです。

できること
  • Web Clipper — 右クリックで Web ページを Markdown として保存
  • Markdown エディタ — ライブプレビュー、LaTeX 数式、Mermaid 図表に対応
  • サイドパネル — Web ページの横に並べて編集（ツールバーアイコンから起動）
データについて
  • ✓ すべてのノートはブラウザ内にローカル保存
  • ✓ クラウド同期はデフォルトで無効（Git バックアップはオプション）
  • ✓ ブラウザのキャッシュ削除ではノートは消えません
  • ⚠ 拡張機能を削除するとデータも消えます — エクスポート機能をご利用ください
