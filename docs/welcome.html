Welcome to your local Markdown workspace

Thanks for installing

A Markdown note-taking app that lives entirely in your browser.

First things first

If you see this dialog

Chrome may show a confirmation dialog right after installation. To use Mark It Down, click "Keep it".

Want to go back to your previous extension? Just turn off Mark It Down at chrome://extensions.

Press Ctrl + T to open a new tab

✓Success: You'll see a note called "Welcome to Mark It Down" — that means it's working

Three ways to use

New Tab

Press Ctrl + T anytime — the editor opens like a notepad.

Side Panel

Click the Mark It Down icon in the toolbar (pin it via the Extensions puzzle icon). Opens the editor beside any webpage. Set up a keyboard shortcut at chrome://extensions/shortcuts.

Web Clipper

Right-click → "Save selection" or "Save page" to clip any webpage as Markdown. The Side Panel opens automatically.

About your data↓
  • ✓Confirmed: All notes stored locally in your browser
  • ✓Confirmed: No cloud sync by default (optional Git backup)
  • ✓Confirmed: Clearing browser cache does not delete your notes
  • ⚠Warning: Removing the extension deletes all data — use the export feature first
What else you can do↓
  • Markdown Editor — Live preview, LaTeX math, Mermaid diagrams
  • Git Sync — Back up to GitHub / GitLab
  • Export — Save as PDF, DOCX, HTML, or PNG
  • 4 Themes — Light, Dark, Parchment, Candlelight
  • Focus Mode — Typewriter scroll for distraction-free writing
